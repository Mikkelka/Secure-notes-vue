# Roadmap (Short)

**Opdateret:** 2026-01-18

Denne roadmap samler ROADMAP + ROADMAP-V2 i en kort prioriteret plan.

---

## ? Allerede implementeret
- Client-side kryptering + SecureStorage
- Google login (Google-only auth)
- Rich text editor (TinyMCE lokal)
- Favoritter, folders, secure folder + PIN
- Trash/recycle bin
- Recent notes
- Export/Import (JSON)
- AI note enhancement

---

## ?? Næste (Høj prioritet / must-have)
1. **Bulk operations (multi-select)**
   - Move/delete/favorite flere noter ad gangen.
2. **Tags**
   - #tags, autocomplete, filter, rename/delete.
3. **Enhanced search & filters**
   - Date range, folder-only search, sort by updated.
4. **Keyboard shortcuts**
   - Ctrl+N, Ctrl+S, Ctrl+F, Esc.

---

## ?? Næste (Medium)
5. **Auto-save + save state**
   - "Saving…" + "Saved at" + draft recovery.
6. **Note linking**
   - [[Note Title]] + backlinks panel.
7. **Performance**
   - Virtual list / pagination for notes.
   - Memoize note counts.

---

## ?? Nice-to-have / senere
8. **Attachments** (images/files, encrypted)
9. **Mobile gestures** (swipe to favorite/delete)
10. **Version history** (diff + restore)
11. **Error tracking** (Sentry)
12. **Tests** (unit + e2e)
